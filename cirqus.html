Introduction

Our team of eight members collaborated to develop a web application for CirQuS, focusing on creating an efficient and intuitive user interface (UI) and user experience (UX) while employing Agile scrum methodologies. We utilized Jira for project management and Confluence for documentation and collaboration. As a web developer, I played a key role in translating these designs into a fully functional application and developing critical features, including an email notification system.

Role and Contributions

In this project, my role as a web developer was centered around translating UI/UX designs into a functional and interactive web application. My key contributions included:

  • Developing an email notification system that allows users to set temperature thresholds for fridge monitoring and receive email alerts when these thresholds are exceeded. This system utilized an API to track and manage temperature data effectively.
  • Assisting in the creation of an interactive graph that visualizes real-time data from the CirQuS API, enhancing the application's ability to provide immediate insights into temperature trends.
  • Working closely with designers to implement their visual and interaction designs onto the web page, ensuring a seamless integration of UI/UX elements.
  • Building and optimizing the application's front-end using modern web technologies such as React.js and Node.js, as well as HTML, CSS, and JavaScript, to ensure responsiveness and accessibility.
  • Collaborating with team members to ensure design consistency and adherence to best practices, while incorporating feedback to iteratively improve the applicationâ€™s functionality and user experience.

Project Overview

The CirQuS Fridge Monitoring web application was designed to provide real-time monitoring of fridge temperature for technology advisors. Using React.js and Node.js, we developed a platform that allows users to track temperature fluctuations and receive timely email notifications for any anomalies. I assisted in creating an interactive graph that visualizes real-time data from the CirQuS API, providing users with immediate insights into temperature trends and ensuring effective management of stored products.

Target Audience

The target audience for the CirQuS Fridge Monitoring web application consists of technology advisors who manage fridge systems across various environments. The application aimed to deliver a reliable and user-friendly solution to support their monitoring tasks and ensure the quality of stored goods.

Results and Impact

The project successfully delivered a feature-rich web application that met the needs of our target audience. Users appreciated the effective email notification system and the intuitive UI/UX design, which facilitated their monitoring tasks and improved their overall experience. The project demonstrated the impact of integrating well-designed features into a cohesive web application to achieve specific user needs and project goals.
